# Core Banking Account Lifecycle Optimization

This repository contains representative Product Management artifacts
supporting a fintech case study focused on optimizing the full account lifecycle
on top of a core banking system (e.g., Temenos T24).

## Contents
- PRD: Product requirements summary
- Metrics: Success metrics and guardrails
- Roadmap: Phased delivery plan
- Risks: Key product and compliance risks
- Diagrams: Lifecycle flow visualization

> Note: Artifacts are representative and anonymized.
